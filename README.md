# Web-Tutor — прототип сервиса онлайн-записи

Интерактивная платформа для поиска репетиторов и онлайн-записи на занятия.

## Функционал
- **Поиск и фильтрация:** Выбор репетитора по учебным предметам.
- **Интерактивная запись:** Динамическая генерация свободных временных слотов в зависимости от выбранной даты.
- **Валидация:** Проверка корректности ввода имени (только буквы) и контактных данных (email или телефон).
- **Адаптивность:** Поддержка мобильных устройств (планшеты и смартфоны).

## Выполненные дополнительные задания
В дополнение к основному ТЗ реализовано:
1. **Выбор длительности занятия:** Возможность выбрать 60, 90 или 120 минут с автоматическим пересчетом итоговой стоимости.
2. **Notification API:** Браузерное уведомление при успешном завершении записи.
3. **LocalStorage:** Все записи сохраняются в браузере пользователя.
4. **Страница "Мои записи":** Отдельный раздел для просмотра истории бронирований с возможностью их отмены.

## Технологии
- **HTML:** Семантическая верстка.
- **CSS:** Flexbox, Grid, адаптивные медиа-запросы.
- **JavaScript:** Работа с DOM, событиями и локальным хранилищем.
